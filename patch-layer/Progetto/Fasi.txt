    Stage    : Color

                * |-> Posterizza
                * |-> Inverti ...
                * |-> Inverti Valore
        * 2) Aggiungere GrayScale per il Color Space.
        3) Integrazione dei layer con Blender.
            3.1) Node Editor:
                 - Il Node Image deve avere la possibilità di poter selezionare
                   tutti i layer o il singolo layer. Es.: Render Layer.
            3.2) Cycles Render:
                 - Il node Image Texture deve essere simile al node Image
                   implementato sopra.
            3.3) Properties Textures:
                 - Se una texture è il risultato di più layer, si deve avere la
                   possibilità di poter applicare al singolo layer tutte le
                   proprietà di una texture(Bump, Normal, Emit, Color, ...)
            3.4) Texture Paint:
                 - Possibilità di poter dipingere con i layer anche nella vista
                   3D. Applicando le proprietà delle texture sul singolo layer.
            3.5) Bake:
                 - Si può avere la possibilità di scegliere quali modalità di
                   Bake fare e portarli tutti in un layer diverso.
